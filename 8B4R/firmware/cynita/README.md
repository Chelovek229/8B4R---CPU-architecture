# Cynita boot screen for 8B4R
Русский / English

Описание
-------
Этот модуль выводит ASCII-логотип `8B4R` и строку:
`CPU architecture by KiG Orgsnization`
и статус `Cynita mode: Off` (по умолчанию).

Нажатие сочетания клавиш Ctrl + C + F12 переключает статус на `Cynita mode: On` и обратно.
(Дальнейшие действия в режиме Cynita будут добавлены по вашему указанию.)

Usage / Использование
-------
1. Сборка:
   - Откройте `firmaware/cynita` проект или добавьте `boot_logo.cpp` в существующий проект Visual Studio.
   - Соберите как консольное приложение (Release/Debug).

2. Запуск:
   - Запустите `boot_logo.exe` в консоли.
   - Экран выведет логотип и инструкции.
   - Нажмите Ctrl + C + F12, чтобы переключить режим.
   - Нажмите ESC для выхода.

Integration / Интеграция
-------
- Чтобы показывать этот экран при загрузке эмулятора/компьютера:
  - Вставьте вызов исполняемого файла в последовательность загрузки эмулятора — например, запустить `boot_logo.exe` перед загрузкой основной прошивки.
  - Встраивание в образ прошивки потребует конвертации бинарника в ROM/Flash и вызова на старте; сообщите желаемый способ интеграции — подготовлю инструкции.

Замечания
-------
- При нажатии Ctrl+C стандартное поведение консоли (SIGINT) подавляется внутри программы, чтобы сочетание могло обрабатываться. Если требуется сохранить стандартное поведение — уберите SetConsoleCtrlHandler.
- Реализация на WinAPI; для кроссплатформенной версии добавлю SDL/SDL2 или терминальную реализацию при необходимости.
